# BloomPetals

BloomPetals est un site web vitrine pour une entreprise spécialisée dans la création de bouquets floraux et la décoration d'événements. Ce projet présente les services, réalisations et l'engagement de l'entreprise envers ses clients.

## Description

Le site web présente :

- L'histoire et la philosophie de BloomPetals.
- Les services proposés : créations de bouquets, décoration florale et ateliers.
- Une galerie de fleurs pour illustrer le travail de l'entreprise.
- Une navigation simple avec des liens vers les différentes sections (Home, Entreprise, Réalisations, Contact).

Le site est conçu pour être clair, esthétique et facile à naviguer pour les visiteurs.

---

## Fonctionnalités

- **Navigation** : Menu de navigation vers les différentes pages du site.
- **Galerie** : Présentation visuelle des bouquets et créations florales.
- **Sections informatives** : "À propos", "Notre Engagement", "Pourquoi nous choisir ?" et "Nos Services".
- **Recherche** : Champ de recherche (prévu pour fonctionnalité future).
- **Design responsive** : Compatible avec différents types d'écrans.

## Installation et utilisation

1. Cloner le dépôt ou télécharger les fichiers.
2. Ouvrir `index.html` dans un navigateur web pour visualiser le site.
3. Modifier le fichier CSS `index.css` pour personnaliser le design si nécessaire.
4. Remplacer les images dans le dossier `images/` pour les mettre à jour.
